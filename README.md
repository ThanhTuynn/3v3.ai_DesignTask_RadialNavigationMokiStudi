# Moki Radial Navigation

An interactive radial navigation prototype designed for the Moki language learning app.
This project explores a clean, center-focused navigation experience using a radial (circular) layout.

---

## 📦 Deliverables

* ✅ Interactive prototype (radial navigation UI)
* ✅ Source code (this repository)
* ✅ AI tools used: Built using v0 and refined manually

---

## 🚀 How to Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project

```bash
npm run dev
```

### 4. Open in browser

```bash
http://localhost:5173
```

## ✨ Features

* Radial navigation centered around "Moki"
* Smooth expand/collapse animations
* Interactive nodes (hover, focus, click)
* Clean UI with no sidebar or traditional navigation
* Dynamic node rendering for easy scalability
